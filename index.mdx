    <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Get started in minutes</h2>
    <p className="text-lg text-gray-600 dark:text-gray-400">Follow these steps to compile and run Dice</p>
  </div>

  <Steps>
    <Step title="Clone the repository">
      Start by cloning the Dice repository from GitHub:
      
      ```bash
      git clone https://github.com/luisedmc/dice.git
      cd dice
      ```
    </Step>
    
    <Step title="Install dependencies">
      Make sure you have a C compiler and Wayland development libraries installed:
      
      ```bash Ubuntu/Debian
      sudo apt-get install build-essential libwayland-dev
      ```
      
      ```bash Fedora
      sudo dnf install gcc wayland-devel
      ```
      
      ```bash Arch Linux
      sudo pacman -S base-devel wayland
      ```
    </Step>
    
    <Step title="Compile the program">
      Compile the source code with the necessary flags:
      
      ```bash
      gcc -o dice dice.c -D_GNU_SOURCE
      ```
      
      <Note>The `-D_GNU_SOURCE` flag is required for `memfd_create` support.</Note>
    </Step>
    
    <Step title="Run the program">
      Execute the compiled binary to create a Wayland window:
      
      ```bash
      ./dice
      ```
      
      You should see a 736x736 pixel window appear with the application displaying the Alya logo. The program will output connection details and protocol events to stderr.
    </Step>
  </Steps>
</div>
